# libdistributed

*Reusable libraries for building distributed systems that actually work under pressure.*

Battle-tested. Peer-reviewed by packet loss. These are the building blocks you reach for when consensus isn't optional and downtime isn't tolerated. No magic; just correct implementations of hard problems, exported and ready to link.

`#include <libdistributed/resilience.h>` — if only it were that easy. We're getting close.
